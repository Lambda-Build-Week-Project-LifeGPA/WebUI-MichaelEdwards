How it works:

As the user you will be provided a list of common habits or you can create your custom habits to add to a list.

The user experience is developed in such a way that you will be able to see an interactable list of your habits. You will be able to mark the ones you achieved that day.

There is an option that will allow you to set reminders for an upcoming habit. You will be able to set how far in advance you want to be notified.

Easy to Use

The layout of the app makes easy for you to use and visualize your progress. Adding a new habit is one click away and removing a habit is just one click away. It is that easy.

Secured Data

You take your data security seriously. We provided a secure connection and only you are allowed to see your data.

Flexible Design

You have the ability to move habits around in a specific manner that you prefer. If you want your most prioritized habits to appear first that is no problem.

Specfic to Each User

We understand that every person is different. We look to always provide an experience that is unique to you.
